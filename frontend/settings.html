📅 Tägliches Ranking-Posting in Discord

Bot postet jeden Tag zur konfigurierten Uhrzeit das Ranking als Embed in einen Discord-Channel. Toleranzfenster ±10 Min nach Soll-Zeit, State wird tagesweise getrackt (kein Doppel-Post nach Service-Restart).

🥇 Daily-Top-3-Hype-Post

Zweiter Auto-Post — kompakt, nur die Top 3 in Inline-Sektionen (Podest-Look). Eigener Channel + eigene Uhrzeit. Toleranzfenster ±10 Min. Ideal für Daily-Hype am Morgen.

🎲 Titel-Pool für Random-Pick

Wenn dieser Pool nicht leer ist, wird bei jedem Top-3-Post ein zufälliger Titel daraus genommen — der „Embed-Titel" oben dient nur noch als Fallback bei leerem Pool. Markdown ist erlaubt (z.B. **fett**).

Pool ist leer — der Fallback-Titel oben wird verwendet.

🎭 Personal-Bedarf — Admin-Channel

Discord-Channel, in den die Personal-Briefings über den „📤 Posten"-Button im Dashboard-Widget gesendet werden. Spielleitung sieht hier pro Mission: Crew, Slot (wann), Stadtteil, Status und den vollen NPC-Plan. Jeder Post ersetzt automatisch den vorherigen für dieselbe Mission.

💬 Reaktions-Antworten (nach 👍 / 👎 / ❌)

Wenn der Boss reagiert, postet der Bot zufällig einen dieser Sprüche als Reply auf den Auftrag. Pool leer → Default „Alles klar, wir melden uns.".

Pool ist leer — Default „Alles klar, wir melden uns." wird verwendet.

⏳ Versager-Sprüche bei abgelaufener Deadline

Wenn ein Auftrag abläuft, ohne dass der Boss reagiert hat, postet der Bot zufällig einen dieser Sprüche im Auftrags-Channel. Markdown wird gerendert (z.B. **fett**). Pool leer → Default „Du hast versagt!".

Pool ist leer — Default „Du hast versagt!" wird verwendet.
