# Die Mittler — Big-Boss-Quest-Geber-Team

Sechs Persönlichkeiten, alle dem Big Boss loyal, aber jede mit eigenem Stil und Schwerpunkt. Gespielt vom User + Team (5–6 Personen). Gangs kennen am Anfang keine von ihnen — sie tauchen über die Tage in RP-Begegnungen auf und übergeben Aufträge.

> Wichtig für die Spielleitung: alle Mittler dienen dem **gleichen Boss**. Loyalität ist zu 100 % geklärt. Was variiert, ist der **Ton**, der **Schwerpunkt** und die **Beziehung zu den Gangs**. So bekommt jede RP-Begegnung eine andere Farbe.

## Workflow: Wie Aufträge fließen

```
        ┌──────────────────────────┐
        │   BIG BOSS (= User)       │
        │   • Plant Story-Beats     │
        │   • Verteilt Briefings    │
        │     an die Mittler        │
        └────────────┬─────────────┘
                     │  (Discord, Voice, Notizen)
                     ▼
   ┌─────────────────────────────────────────┐
   │   5–6 MITTLER (User + Team)              │
   │   Miguel · Maklerin · Pater · Fixer ·    │
   │   Witwe · Skrupellose                    │
   │   • Übersetzen Briefings in eigenen Stil │
   │   • Übergeben in RP-Begegnungen          │
   │   • Sammeln Feedback aus den Gangs       │
   └────────────┬────────────────────────────┘
                │  (In-Game-RP, Discord-DMs)
                ▼
   ┌──────────────────────────────────────────┐
   │   21 CRIME-CREWS + 13 ZIVIL-FIRMEN        │
   │   • Empfangen Auftrag                     │
   │   • Reagieren mit 👍/👎/❌ im System       │
   │   • Boss-Feedback im Info-Channel         │
   └──────────────────────────────────────────┘
```

**Was der User (= Big Boss) konkret tut:**
- **Vorbereitung:** plant Story-Beats für die kommenden Tage (Master-Plot aus EVENT_TIMELINE.md)
- **Briefing an Mittler:** verteilt vor jedem großen Tag (2, 4, 7, 9, 10) das Voice-Over-Skript aus EVENT_BRIEFINGS_MASS.md an die Mittler — die wählen ihren Stil, übersetzen es eigenständig
- **Zwischen den großen Tagen:** verteilt Themen / lose Auftrags-Ideen an einzelne Mittler, die selbständig RP-Begegnungen organisieren
- **Reaktionen sammeln:** liest Discord-Channels, Info-Channels, hört Mittler-Feedback aus dem In
